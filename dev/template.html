Wander Console Network

Wander is a small, decentralised, self-hosted web console that lets visitors explore websites and pages recommended by a community of independent personal site owners. Learn more at codeberg.org/susam/wander. This page shows a snapshot of Wander consoles and recommended pages from the Wander Console Network discovered by crawling the network from a set of known consoles. The data on this page is also available as JSON: wcn.json. This snapshot was generated on @date.
